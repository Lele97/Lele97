|---|---|
| **Frontend** | React · Vite · TypeScript |
| **Backend** | Java · Spring Boot · REST API |
| **Messaging** | Apache Kafka · NATS |
| **Database** | PostgreSQL |
| **DevOps** | Docker · GitHub Actions · PM2 |
